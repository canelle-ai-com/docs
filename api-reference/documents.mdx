The Documents API lets you manage your knowledge base programmatically — add new sources so your agent has up-to-date information, list everything that's currently indexed, and remove outdated or incorrect content. Changes take effect as soon as a document's status transitions to `ready`, which typically happens within a few seconds for small pages and a few minutes for longer content.

---

## POST /v1/documents

Add a new document to your knowledge base by providing a URL to scrape or plain text content to index directly.

**Endpoint**

```
POST https://api.canelleai.com/v1/documents
```

### Request Body

<ParamField body="project_id" type="string" required>
  The ID of the project to add this document to.
</ParamField>

<ParamField body="type" type="string" required>
  The type of content to index. Accepted values: `url`, `text`.
</ParamField>

<ParamField body="url" type="string">
  The URL to scrape and index. Required when `type` is `url`.
</ParamField>

<ParamField body="content" type="string">
  Plain text content to index directly. Required when `type` is `text`.
</ParamField>

<ParamField body="title" type="string">
  A human-readable name for this source. If omitted, Canelle AI uses the page title for URLs or the first line of content for text sources.
</ParamField>

### Example Request — Add a URL

```bash curl
curl -X POST https://api.canelleai.com/v1/documents \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "project_id": "proj_01HX9Z2K3M",
    "type": "url",
    "url": "https://acme.com/help/shipping",
    "title": "Shipping Policy"
  }'
```

### Example Request — Add Plain Text

```bash curl
curl -X POST https://api.canelleai.com/v1/documents \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "project_id": "proj_01HX9Z2K3M",
    "type": "text",
    "content": "Our support team is available Monday to Friday, 9am–6pm EST. Contact us at support@acme.com.",
    "title": "Support Hours"
  }'
```

### Response Fields

<ResponseField name="id" type="string">
  The unique ID of the newly created document. Save this if you need to delete the document later.
</ResponseField>

<ResponseField name="status" type="string">
  The current indexing status of the document. Possible values:
  - `processing` — the content is being scraped and indexed
  - `ready` — indexing is complete and the document is available to your agent
  - `error` — indexing failed; check the `error_message` field for details
</ResponseField>

<ResponseField name="title" type="string">
  The display name of the document.
</ResponseField>

<ResponseField name="type" type="string">
  The source type: `url` or `text`.
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the document was created.
</ResponseField>

## GET /v1/documents

List all documents currently in a project's knowledge base.

**Endpoint**

```
GET https://api.canelleai.com/v1/documents?project_id=proj_xxx
```

### Query Parameters

<ParamField query="project_id" type="string" required>
  The ID of the project whose documents you want to list.
</ParamField>

### Example Request

```bash curl
curl "https://api.canelleai.com/v1/documents?project_id=proj_01HX9Z2K3M" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Example Response

```json
{
  "documents": [
    {
      "id": "doc_01HX9Z2K3M",
      "title": "Shipping Policy",
      "type": "url",
      "url": "https://acme.com/help/shipping",
      "status": "ready",
      "created_at": "2024-03-15T10:30:00Z"
    }
  ]
}
```

## DELETE /v1/documents/{id}

Permanently remove a document from your knowledge base.

**Endpoint**

```
DELETE https://api.canelleai.com/v1/documents/{id}
```

### Path Parameters

<ParamField path="id" type="string" required>
  The ID of the document to delete. Retrieve this from the list endpoint or from the response when the document was created.
</ParamField>

### Example Request

```bash curl
curl -X DELETE https://api.canelleai.com/v1/documents/doc_01HX9Z2K3M \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Example Response

```json
{
  "deleted": true
}
```

<Warning>
  Deletion is permanent. The document is removed from the index immediately and your agent will no longer be able to reference it in answers. This action cannot be undone.
</Warning>
